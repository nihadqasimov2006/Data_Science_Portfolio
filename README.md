## 🗂️ Projects

### 1. 📉 Telco Customer Churn Prediction
*Binary classification — predict which telecom customers will churn before they leave.*

- **Dataset:** IBM Telco Customer Churn (7,043 rows, 21 features)
- **Problem:** Class imbalance (~27% churn) — handled with SMOTE
- **Models:** Random Forest, XGBoost (scale_pos_weight + threshold optimization)
- **Best result:** ROC-AUC: ~0.84 · F1 (churn class): ~0.69
- **Key insight:** Tenure is the single strongest predictor — long-tenured customers rarely churn.
- **Techniques used:** EDA, feature selection (Chi2 + ANOVA), SMOTE, MinMaxScaling, threshold optimization, ROC curve comparison.

### 2. 🌸 Iris Dataset — KMeans Clustering
*Unsupervised learning — cluster iris flowers without using labels.*

- **Dataset:** Iris (150 samples, 4 features)
- **Problem:** Find natural groupings using KMeans
- **Evaluation:** Elbow method, Silhouette score
- **Key insight:** Setosa is perfectly separable; Versicolor and Virginica overlap in feature space.
